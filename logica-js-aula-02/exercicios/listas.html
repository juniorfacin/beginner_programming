Desafio 03 - Hora da Prática

  1. Crie uma lista vazia, com o nome listaGenerica.

  2. CCrie uma lista de linguagens de programação chamada linguagensDeProgramacao com os seguintes elementos: 'JavaScript','C','C++', 'Kotlin' e 'Python'.

  3. Crie uma função que converta um valor em dólar, passado como parâmetro, e retorne o valor equivalente em reais. Para isso, considere a cotação do dólar igual a R$4,80.

  4. Adicione à lista linguagensDeProgramacao os seguintes elementos: 'Java', 'Ruby' e 'GoLang'.

  5. Crie uma lista com 3 nomes e exiba no console apenas o primeiro elemento.

  6. Crie uma lista com 3 nomes e exiba no console apenas o segundo elemento.

  7. Crie uma lista com 3 nomes e exiba no console apenas o último elemento.
